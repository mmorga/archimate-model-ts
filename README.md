# Archimate Model

# What is this?

This is a Typescript Module to support loading ArchiMate models.

# How do I use it?

## Install your dependencies

Run `npm install`

## Building the repo

```sh
npm run build
```

## Type-checking the repo

```sh
npm run type-check
```

And to run in `--watch` mode:

```sh
npm run type-check -- --watch
```

### Run the build task

```
npm run bundle
```
